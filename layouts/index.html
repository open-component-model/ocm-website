Why Choose OCM?

📦 One Model for All Artifacts

A single, machine-readable format for containers, Helm charts, binaries, and data—everything you deliver.

🔒 Security & Integrity

Components including their artifacts are signed. Verify provenance at every step.

🌐 Universal Delivery

Transfer and deploy components to any environment, public and private clouds, on-premises and even air-gapped environments.

🔗 End-to-End Traceability

OCM Coordinates stitch together all lifecycle phases. Correlate build, test, deploy and security metadata into one global view.

⚙️ GitOps-Ready Automation

OCM Controllers and OCM CLI plug straight into your CI/CD pipelines — no custom glue code needed.

How OCM Works

Describe

Define component versions in code. Attach access methods and lifecycle metadata.

Sign

Add cryptographic signatures.

Transport

Works across boundaries — public cloud, on-prem, or air-gapped. Immutable metadata ensures tamper-proof delivery.

Deploy
